OASIS Innovation: What's Actually Novel

Key Statistical Advances

Finite-Sample Validity

  • Closed-form P-value bounds
  • Valid for ANY sample size
  • No asymptotic approximations
  • Mathematically guaranteed Type I error control

Structured Noise Robustness

  • Bilinear form averages out noise
  • Focuses on biological signal
  • Robust to overdispersion
  • Built-in interpretability

Handling Unwanted Variability: The RUV Problem

The Challenge in High-Throughput Data

Sources of Unwanted Variation

  • Batch effects - Different sequencing runs
  • Technical noise - Equipment variation
  • Overdispersion - Biological sampling noise
  • Contamination - Mixed samples
  • Library prep - Protocol differences

OASIS Solution
